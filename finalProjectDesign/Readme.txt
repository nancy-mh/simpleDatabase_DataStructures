*******************************************************
*  Description of the program
*******************************************************

The program “simpleDatabaseSystem” uses binary search trees and vectors as the main data structures to produce a small database that allows users to add, delete, search, sort, modify and print out reports.

*******************************************************
*  Source files
*******************************************************

Name:  main.cpp
   Main program.  This calls main menu function which is the driver function of the database. It also has implementations of add, delete, search, sort, modify and print to file functions.

Name: BinarySearchTrees.cpp
implements the class ActorActressBSTree and PicturesBSTreefunctions functions, findNode, addNode, deleteNode, and inorderTraversal that work with the binary search tree data structure.

Name: BinarySearchTrees.h
Contains the definition for the class ActorActressBSTree and PicturesBSTree. 

Name: Vectors.cpp
Implements ActorActress and Pictures classes functions, exactSearch, containsSearch, sort, writeToFile, and writeCSV functions. 
WriteToFile function produces a table and tabs formatted report. Formatting based on string length needs to be implemented. 

Name: Vectors.h
Contains the definition for the class ActorActress and Pictures.

Name: Node.h
Implements and defines NodeActorActress and NodePictures classes.

*  Circumstances of programs
